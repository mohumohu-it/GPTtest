# Mini Spire (Text Prototype)

スレイザスパイア風の**テキストベース・ローグライクデッキ構築**をブラウザで遊べるようにした試作です。

## 起動方法

`index.html` をブラウザで開くだけで動きます。

## 実装済み（簡易版）

- マップイベント（戦闘 / 休憩 / ショップ）
- ターン制カードバトル
- 山札・手札・捨て札の循環
- 戦闘報酬でカードを追加
- 10階層でゲームクリア

## 使用技術

- HTML
- JavaScript（フレームワークなし）
- Sass（`styles.scss`） + 生成済み `styles.css`
