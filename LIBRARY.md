## イベントライブラリ

| イベント名 | 主な選択肢 | 主な結果 |
|---|---|---|
| 忘れられた宝箱 | 開ける / 開けない | コモンレリック獲得 or スルー |
| 回復の泉 | HP20%回復 / 最大HP10%アップ | 生存力強化 |
| 血の祭壇 | 血を捧げる / 立ち去る | HP減少と引き換えにGold |
| 行商人の落とし物 | 拾う / 無視する | Gold + 防護札 |
| 古びた偶像 | 触れる / 引き返す | 脱力を受ける代わりにGold |
| 静寂の書庫 | 読む / 立ち去る | アンコモン/レアカード獲得 |


## 祝福ライブラリ

ゲーム開始時に3択で1つだけ選択。

| 祝福内容 | 効果 |
|---|---|
| 最大HP+8 | 最大HPを8増やし、現在HPも8回復 |
| 100G獲得 | 所持金+100 |
| レアカード1枚獲得 | ランダムなレアカードをデッキに追加 |
| ランダムレリック1個獲得 | 未所持のレリックを1つ獲得 |
| HPを全回復 | 現在HPを最大まで回復 |
